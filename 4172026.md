## Individual Fixes

### Fixed Gate Pass Management Module

Several improvements were applied to the Gate Pass Management Module.

#### Changes Made

- Added a combination of filter and group for selecting the route ID when issuing a gate pass:
  - Filter: `https://v9.primereact.org/listbox/#filter`
  - Group: `https://v9.primereact.org/listbox/#group`
- Reworked the `expiry_hrs` logic:
  - Previous setup: 24-hour expiry
  - New setup:
    - Start Time: 8:00 AM
    - End Time: 11:00 PM
- Added automatic handling for gate passes issued past 11:00 PM:
  - Marked as `Expired` in `trans_status`
  - Marked as `Cancelled` in `status`
